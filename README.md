# xfce4-systemload-plugin

A system load plugin for the Xfce4 desktop environment. It displays the
current CPU load, the memory in use, the swap space and the system uptime in
the Xfce4 panel.

### Installation

From source code repository: 

    % cd xfce4-systemload-plugin
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf xfce4-systemload-plugin-<version>.tar.bz2
    % cd xfce4-systemload-plugin-<version>
    % ./configure
    % make
    % make install
